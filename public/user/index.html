Odzreshop Selalu Ada Untuk Bisnis Kamu

Pertanyaan yang sering ditanyakan seputar layanan kami

Apa itu Odzreshop API?
Odzreshop API adalah platform layanan penyedia antarmuka pemrograman aplikasi (API) yang memudahkan developer dan pebisnis untuk mengintegrasikan layanan seperti QRIS Payment, Cek ID Game, dan WA Gateway ke dalam sistem mereka sendiri.
Apakah Ada biaya bulanan di Odzreshop?
Ya, kami memiliki sistem berlangganan yang sangat terjangkau. Biaya bulanan mulai dari Rp 5.500/bulan saja untuk memberikan Anda akses penuh ke berbagai fitur layanan API unggulan kami.
Apakah Metode QRIS tersedia?
Ya, kami menyediakan layanan konversi QRIS statis menjadi QRIS dinamis lengkap dengan nominal custom dan sistem deteksi pembayaran otomatis (Auto-polling & Webhook).
Apakah Tersedia Realtime Settlement?
Tergantung pada jenis layanan QRIS yang Anda integrasikan. Untuk layanan integrasi via akun e-wallet Anda sendiri, dana akan masuk secara realtime ke akun e-wallet/merchant Anda.
Berapa Lama Proses Aktivasi Akun?
Aktivasi akun dilakukan secara instan. Setelah Anda mendaftar, Anda bisa langsung membuat API Key di dashboard dan mulai mencoba layanan kami.
Info Lengkap Integrasi API?
Anda dapat melihat tata cara integrasi, contoh request, dan response dari masing-masing endpoint langsung melalui halaman API Docs kami yang interaktif.
